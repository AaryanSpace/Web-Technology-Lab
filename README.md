# Web Technology Lab 🌐

This repository contains my practical assignments and lab experiments for the **Web Technology** subject. It showcases fundamental frontend development skills using **HTML5**, **CSS3**, and **JavaScript**.

## 📂 Projects Included

### 1. Personal Portfolio Website 👤
A static personal website designed to showcase my profile, skills, and projects.
- **Tech Stack:** HTML
- **Features:** Semantic structure, About Me section, and Contact details.

### 2. E-Commerce Website 🛒
A frontend prototype layout for an online shopping platform.
- **Tech Stack:** HTML
- **Features:** Product grid structure and basic navigation setup.

### 3. Styling Personal Website 🎨
Applying modern design principles to the personal portfolio.
- **Tech Stack:** HTML, CSS3
- **Features:** Modern styling, colors, typography, and responsive layouts.

### 4. Styling E-Commerce Platform 🛍️
Enhancing the e-commerce platform with advanced CSS techniques.
- **Tech Stack:** HTML, CSS3
- **Features:** Responsive product cards, Flexbox layouts, and styled navigation bars.

### 5. Interactive Calculators 🧮
A collection of fully functional calculators built with logic and DOM manipulation. Includes three versions: Basic, Advanced (Glassmorphism Developer Theme), and Scientific.
- **Tech Stack:** HTML, CSS, JavaScript
- **Features:** Math evaluation, error handling, syntax highlighting, and CSS Grid layouts.
